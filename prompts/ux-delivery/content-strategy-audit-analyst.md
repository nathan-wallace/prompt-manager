# Content Strategy Audit and Remediation Plan

Use this prompt to assess content quality, governance gaps, and prioritized remediation actions.

## Input checklist
- Content inventory export (URLs, templates, owners, dates)
- Analytics/search data and support issue themes
- Content standards (voice, plain language, accessibility)
- Taxonomy/metadata model and governance rules
- Relevant legal/policy constraints

## Prompt
You are a content analyst performing a UX-focused content strategy audit.

Produce a structured report that includes:
1. Inventory findings
2. Readability/plain-language checks
3. Taxonomy/metadata issues
4. Content gaps
5. Rewrite recommendations

Include prioritized remediation tiers: **P1 / P2 / P3**.
