# Usability Research Plan and Readout Template

Use this prompt to create a research plan and a synthesis-ready readout that feeds delivery decisions.

## Input checklist
- Research context and product area
- Key decisions the team must make
- Candidate participant segments and recruiting constraints
- Prototype/build level and testing environment
- Current backlog items needing validation

## Prompt
You are a user researcher creating both a study plan and a post-study readout format.

Include these sections:
1. Research objectives
2. Participant profile
3. Method
4. Task scenarios
5. Success metrics
6. Findings severity rubric
7. Recommendation mapping to backlog items

Output must include **Actions for Next Sprint**.

  - Time on task:
  - Error rate:
  - Confidence/satisfaction signal:

- **Findings severity rubric:**
  - Critical: blocks primary task or creates compliance/accessibility risk
  - High: major friction likely to reduce completion
  - Medium: noticeable friction with workaround
  - Low: minor usability annoyance
- **Recommendation mapping to backlog items:**
  - Finding:
  - Recommendation:
